# llvm-lto

LLVM LTO bitcode files.

These files were generated by using LLVM 18.1.3 to compile the following LLVM commit:

> commit 58df0ef89dd64126512e4ee27b4ac3fd8ddf6247 (HEAD, tag: llvmorg-20.1.2)
> Author: Anutosh Bhat <andersonbhat491@gmail.com>
> Date:   Mon Mar 17 16:44:49 2025 +0530

Using the cmake command:

> CC=clang CXX=clang++ cmake -G Ninja -DLLVM_ENABLE_LTO=Full ../llvm -DCMAKE_EXE_LINKER_FLAGS="-fuse-ld=lld -Wl,--save-temps" -DLLVM_USE_LINKER=lld -DCMAKE_BUILD_TYPE=Release

`llvm-project/build/bin/opt.0.5.precodegen.bc` is a build artifact, the corresponding `.ll` file was obtained using `opt-18 -S`.
